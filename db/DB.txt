SET AUTOCOMMIT=0 - отключает автокоммит
start transaction - начинает транзакцию
SAVEPOINT my_savepoint; - создание сейвпоинта для транзакции. (затем можно будет сделать ROLLBACK TO SAVEPOINT name)
commit - завершает транзакцию
rollback - откатывает изменения
    Выключение сервера; в этом случае откат транзакции будет выполнен автоматически при возобновлении работы сервера.
    Выполнение еще одной команды start transaction, в результате чего происходит фиксация предыдущей транзакции
    Преждевременное завершение транзакции сервером, который выявил взаимоблокировку (deadlock - когда 2 транзакции сослались на одни и теже данные и ждут завершения друг друга) и решил, что виновна в этом данная транзакция. В этом случае будет выполнен откат транзакции и пользователь получит сообщение об ошибке
